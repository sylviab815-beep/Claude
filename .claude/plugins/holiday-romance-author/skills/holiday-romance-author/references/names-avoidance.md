## How to Use This List

**Pre-Writing (Phase 3):**
1. Extract all character names from your outline
2. Check each name against this list
3. If a name appears on the list, replace it with a fresh alternative
4. Keep a replacement log (Original Name → New Name)

**During Writing (Phase 4):**
- Reference the replacement log to ensure consistency
- If you introduce a new character mid-writing, check their name against this list
- Replace any matches immediately

**During Revision (Phase 5):**
- Re-verify all character names against this list
- If edits introduced new names, check them
- Ensure no names were accidentally reverted to originals

## How to Find Fresh Name Alternatives

### Methods for Generating New Names

**1. Historical figures or literature (outside romance):**
- "Austen" (after Jane Austen, but used as first name)
- "Hadley" (from Ernest Hemingway)
- "Rowan" (from various mythologies, but literary)

**2. Geography or place names:**
- "Piper" (place-adjacent, modern feel)
- "Zoey" (different take on nature)
- "Lincoln" (place name with modern appeal)

**3. Word names with double meanings:**
- "Blake" (pale/dark)
- "Kelsey" (island)
- "Mercer" (merciful)

**4. Surname-as-first-name (modern trend):**
- "Parker"
- "Avery"
- "Quinn"
- "Riley"

**5. Non-English origins (authentic to character background):**
- Spanish: "Miguel," "Sofía," "Carlos," "Lucia"
- Italian: "Marco," "Giulia," "Leonardo," "Francesca"
- Irish: "Cormac," "Siobhan," "Liam," "Fiona"
- Welsh: "Ceredwyn," "Gareth," "Branwen," "Iorwerth"
- Scandinavian: "Soren," "Kaja," "Erik," "Greta"

**6. Check against popular culture to avoid recent franchises:**
- Don't use character names from recent bestselling romance (last 3-5 years)
- Don't use character names from popular TV shows/movies

## Red Flags for Character Names

**Avoid:**
- ✗ Names that are literally the holiday (Santa, Claus, Scrooge for Christmas)
- ✗ Names that are literally holiday-related concepts (Noel, Holly, Evergreen)
- ✗ Names that are puns (Faith & Hope for Christmas, Cupid for Valentine's)
- ✗ Names that describe the character's role (Hunter for a hunter, Sage for a wise person)
- ✗ Names that are overly trendy (if a name went viral in 2023, probably overused now)
- ✗ Names that are too similar to famous romance authors' character names
- ✗ Gendered names used opposite of typical gender if not integral to character identity

**Safe:**
- ✓ Names with no holiday connection
- ✓ Names that are authentic to character background/ethnicity
- ✓ Names that are slightly off-beat but not ridiculous
- ✓ Names that work in dialogue (easy to pronounce, not too many syllables)
- ✓ Names that are different enough from each other (Don't have multiple characters with A-names or similar sounds)

## Testing Your Character Names

**Question 1: Is this name on the built-in list?**
- If yes, replace it

**Question 2: Is this name a holiday-specific overused name?**
- If yes, replace it

**Question 3: Does this name sound like it belongs to a Hallmark character?**
- If yes, probably replace it

**Question 4: Could I find this exact character (name + basic description) in a bestselling romance published in the last 5 years?**
- If yes, probably replace it

**Question 5: Is the name serving the character, or is the character serving the name?**
- Examples:
  - "Holly for Christmas" = character serving name (avoid)
  - "Hadley, who happens to be in a Christmas story" = name serving character (good)

**Question 6: Would this name feel authentic in natural speech?**
- Can you imagine a real person calling this character by this name?
- Or does it sound like a character from a romance novel?

## Name Diversity

**Ensure character names reflect diversity of the real world:**
- Don't default all female names to "pretty/soft" names
- Don't default all male names to "strong/hard" names
- Include names from various cultural backgrounds if authentic to character
- Include names that span age ranges (don't assume young female character = trendy name; young male character = classic name)

**Example diverse character set:**
- "Marcus" (traditional) paired with "Priya" (Indian origin) for couple
- "Keisha" (African-American origin) paired with "Daniel" (Scandinavian origin) for couple
- "Alejandro" (Spanish) as secondary character
- "Chen" (Chinese) as secondary character

## Final Check Before Writing

**Create a character name list:**
1. List every named character in your outline
2. Check each name against this document
3. Note any matches
4. Replace matches with alternatives
5. Verify all names are distinct enough (no two names starting with same letter if possible, or if necessary, ensure they sound very different when spoken)

**Lock in names:**
- Once names are verified, keep a reference document
- Use this document throughout writing to ensure consistency
- Add new character names immediately to the document and verify against the list

This guardrail prevents the #1 mistake in holiday romance: characters with names so holiday-specific or overused that they feel like Hallmark-movie stereotypes before the first chapter even begins.
