# Manuscript Formatting Template (.docx)

This template ensures the final manuscript is properly formatted for KDP submission or professional formatting.

---

## Document Setup

**Font:** Times New Roman 12pt
**Line Spacing:** 1.5
**Paragraph Spacing:** 0 pt before, 0 pt after (line spacing handles spacing)
**Alignment:** Left
**First Line Indent:** 0.5" for body text paragraphs (not scene breaks or special formatting)
**Page Size:** 8.5" x 11" (standard US Letter)
**Margins:** 1" on all sides

---

## Front Matter

### Title Page

Centered, approximately 1/3 down the page:

```
[TITLE IN ALL CAPS]

[Subtitle if applicable]

By

[Author/Pen Name]
```

No page number on title page.

### Copyright Page (Page 2)

Left-aligned, standard copyright information:

```
Copyright © [Year] [Author/Pen Name]

All rights reserved. No part of this publication may be reproduced,
distributed, or transmitted in any form or by any means, including
photocopying, recording, or other electronic or mechanical methods,
without the prior written permission of the publisher, except in the
case of brief quotations embodied in critical reviews and certain other
noncommercial uses permitted by copyright law.

This is a work of fiction. Names, characters, places, and incidents are
either the product of the author's imagination or are used fictitiously,
and any resemblance to actual persons, living or dead, businesses,
companies, events, or locales is entirely coincidental.

First Edition: [Month Year]

Published by [Publisher Name or "Self-published"]

ISBN: [If applicable]

Cover design by [Designer Name or "Self-designed"]
```

Page number ii (lowercase Roman numeral).

### Table of Contents (Page 3+)

Format:

```
CONTENTS

Chapter 1: [Chapter Title] .......................... 1

Chapter 2: [Chapter Title] .......................... 12

Chapter 3: [Chapter Title] .......................... 25

[... and so on]

Author's Note .......................... [page number]

About the Author .......................... [page number]
```

Each chapter title linked to its page number (or manual page numbers if not using automatic TOC).

Page numbers in lowercase Roman numerals (iii, iv, etc.) for front matter.

## Main Text

### Chapter Headers

Start each chapter on a new page. Format:

```
[Page break before]

CHAPTER ONE

[Chapter title optional, can be just "CHAPTER ONE"]

[Body text begins here, first paragraph NOT indented on first line]
```

Or with chapter titles:

```
[Page break before]

CHAPTER ONE
The First Meeting

[Body text begins]
```

Chapter headers in ALL CAPS, centered, bold optional (but standard in romance).

First paragraph after chapter header: no first-line indent (optional, some publishers prefer indent on all paragraphs).

### Body Text

Standard paragraph formatting:
- Times New Roman 12pt
- 1.5 line spacing
- 0.5" first-line indent
- Left-aligned
- One space after periods (modern standard)

Dialogue:
- Each speaker's dialogue on new line
- Standard quotation marks (" " not " ")
- Action beats tagged to dialogue if same speaker:

```
"This is dialogue," she said, turning away from him.

He reached for her hand. "I can't let you leave like this."

"You have to," she whispered.
```

### Scene Breaks

Between scenes in the same chapter, use a single centered line break with three asterisks:

```
[End of scene paragraph]

*     *     *

[New scene begins]
```

Or use this format (centered):

```
~~~
```

**Spacing:** 1 blank line before scene break, 1 blank line after.

### Emphasis

- **Bold** for emphasis (used sparingly)
- *Italics* for internal monologue, thoughts, or foreign language words
- Do NOT use underline (outdated for digital manuscripts)

Example:
```
She stared at the letter, her heart racing. *What does he want?* It had
been three years.

The handwriting was unmistakably his.
```

### Formatting Special Elements

**Italics for:**
- Internal monologue/thoughts
- Foreign language words (if not translated)
- Emphasized words
- Book titles, movie titles, song titles (if referenced)

```
She'd always loved Christmas, but this year the holiday felt heavy.
*Was she making a mistake?*

The French phrase he'd whispered—"Je t'aime"—still echoed.
```

**Bold for:**
- Emphasized words (use sparingly, maybe 1-2 per chapter)
- Section breaks that aren't scene breaks (if applicable)

### Dialogue Formatting

**Correct dialogue formatting:**

```
"This is how dialogue looks," she said.

"Each speaker gets a new line," he replied.

"Internal dialogue is italicized," she thought.

"Dialogue with action beats," he said, then kissed her, "continues
without new paragraph breaks."
```

**Avoid:**
- Single quotation marks (' ')
- Multiple speakers in one paragraph
- Dialogue tags other than: said, asked, replied, whispered, shouted (use sparingly)

### Paragraph Breaks

Single line break between paragraphs (default for 1.5 line spacing).

Do NOT add extra space between paragraphs; let the line spacing handle it.

## Back Matter

### Author's Note (if applicable)

New page, formatted as chapter header:

```
[Page break]

AUTHOR'S NOTE

[Text in standard paragraph format]
```

Optional section; can be omitted if author has nothing to say.

### About the Author

New page, formatted as chapter header:

```
[Page break]

ABOUT THE AUTHOR

[Author bio, 100-150 words]

[Author website/social media if applicable]
```

Standard paragraph format. Can include a photo (optional; space for it).

### Additional Back Matter (optional)

**Excerpt from Next Book:**
If this is a series, include first chapter (or excerpt) of next book.

```
[Page break]

PREVIEW: [Title of Next Book]

[Chapter 1 excerpt or first few pages]
```

**Call-to-Action:**
End with prompt for reader to connect:

```
Want to know when the next book releases? Sign up for my newsletter at
[website].

Follow me on [social media] for updates, bonus scenes, and behind-the-scenes
content.

Thank you for reading!
```

## Common Formatting Issues to Avoid

**Issue:** Double spaces after periods
- **Fix:** Use single space after periods (modern standard)

**Issue:** Multiple spaces between paragraphs
- **Fix:** Use single line break; let 1.5 spacing handle spacing

**Issue:** Inconsistent font or font sizes
- **Fix:** Maintain Times New Roman 12pt throughout

**Issue:** Underlined text
- **Fix:** Use italics or bold instead

**Issue:** Multiple spaces before dialogue
- **Fix:** Single space/line break before dialogue tags and dialogue

**Issue:** Widows/orphans (single lines at top/bottom of page)
- **Fix:** Let your word processor auto-adjust; minor widows/orphans are acceptable in self-published work

**Issue:** Page numbers in body text
- **Fix:** Use document's header/footer feature to insert page numbers automatically

---

## Header/Footer Setup

**Headers:**
- Leave blank (no book title/author name in header for self-published)
- Or include book title on even pages, author name on odd pages (optional)

**Footers:**
- Page numbers right-aligned in footer
- Front matter: lowercase Roman numerals (i, ii, iii)
- Main text: Arabic numerals (1, 2, 3)

Alternatively: Center page numbers in footer.

## Final Manuscript Preparation

**Before Final Export:**

1. Remove all tracked changes
2. Check for:
   - Consistent formatting throughout
   - No double spaces after periods
   - All dialogue properly formatted
   - Scene breaks consistent
   - Chapter headers match table of contents
   - Page breaks between chapters
   - No blank pages (except intentional ones)
3. Do a final proofread
4. Save as .docx (Microsoft Word format) for KDP

**Export to .docx:**
- File → Save As → Format: Word Document (.docx)
- Name: `YourName-HolidayRomance-Final.docx`

This manuscript is now ready for:
- Direct KDP upload
- Submission to professional formatter
- Print-on-demand service
- Traditional publisher submission (if applicable)

## Quick Reference: Page Break Locations

Use page breaks at these points:

1. Before Title Page (if document starts with text)
2. After Title Page (before Copyright Page)
3. After Copyright Page (before Table of Contents)
4. After Table of Contents (before Chapter 1)
5. Before each chapter header
6. Before Author's Note (if included)
7. Before About the Author
8. Before any back matter sections

**Navigation:** Use Ctrl+Return (or Cmd+Return on Mac) to insert page breaks automatically.
