# KDP Info Document Template

Generate a companion .docx file with Amazon KDP metadata for each manuscript.

## Document Structure

Create a .docx with the following sections:

### 1. Book Information
- **Title**: [Book Title]
- **Subtitle**: [Optional subtitle suggestion]
- **Author**: [Author name — use whatever the user specified, or their pen name]
- **Series Name**: [If part of a series]
- **Book Number**: [If part of a series]

### 2. Amazon Book Description
Write a compelling book description, max 4000 characters. Structure:
- Hook line (1 sentence that grabs attention)
- Setup paragraph (the world, the sleuth, the situation)
- The inciting incident (the murder/mystery)
- Stakes and complications (what makes this mystery different)
- Tease the ending without spoiling it
- End with a call-to-action question or promise

Use HTML formatting allowed in Amazon descriptions: `<b>`, `<i>`, `<br>`, `<h2>`, `<ul>`, `<li>`

### 3. Amazon Keywords (7 phrases)
Seven keyword phrases optimized for the specific cozy subgenre. Each should be 2-4 words. Mix broad and specific:
- 2 broad genre keywords (e.g., "cozy mystery," "paranormal cozy mystery")
- 2 trope/theme keywords (e.g., "amateur sleuth mystery," "small town mystery")
- 2 specific niche keywords (e.g., "bakery cozy mystery," "cat familiar mystery")
- 1 comp/readership keyword (e.g., "books like [comp author]")

### 4. BISAC Categories (3 suggestions)
Three BISAC codes that fit the book, ordered by relevance.

### 5. Pricing Suggestion
- Ebook price recommendation with reasoning
- Print price recommendation (if applicable)
- KU recommendation (yes/no with reasoning)

### 6. Comparable Titles (3-5)
Real published books that share DNA with this manuscript. For each:
- Title and author
- Why it's comparable (specific tropes, tone, or audience overlap)

### 7. Target Audience
Who this book is for — be specific:
- Age range
- Reading preferences
- What they're coming from (other series, other genres)
- What draws them in

### 8. Cover Image Prompt
A detailed, ready-to-use AI image generation prompt. Must include:
- The exact book title as rendered text
- Author name as rendered text
- Illustrated cozy mystery style (whimsical, bright, inviting)
- Specific scene or character elements from the book
- Color palette appropriate to the subgenre
- Composition notes for standard ebook cover (1600x2560px, 2:3 ratio)
- If paranormal: the familiar animal prominently featured
- If culinary: food/kitchen elements prominently featured
- Typography suggestions (font style, placement)

### 9. Series Marketing Notes (if applicable)
- Reading order
- How this book connects to the series
- Whether it works as a standalone entry point
- Suggested series pricing strategy
