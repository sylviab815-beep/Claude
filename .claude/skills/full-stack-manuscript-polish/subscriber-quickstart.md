# Subscriber Quickstart

## What this is

The Full-Stack Manuscript Polish Skill is a reusable editorial engine you can install once and use after every book.

It is designed to help an LLM run a structured editorial process instead of giving vague feedback or rewriting your manuscript into beige soup.

## How to use it

1. Upload or install the skill.
2. Upload your manuscript, chapter list, or chapter text.
3. Tell the AI your genre, POV, tense, current word count, target word count, and whether the book is standalone or series.
4. Run one pass at a time.
5. Apply fixes before moving to the next pass.

## Starter command

Use the Full-Stack Manuscript Polish Skill. Run Pass 1: Structure Check on this manuscript. Do not line edit.

## Command Menu

Run Pass 1: Structure Check. Do not line edit.

Run Pass 2: Expansion Check. Tell me where the manuscript is thin and what scenes should be added.

Run Pass 3: Chapter Openings. Flag openings that start with recap, routine, summary, or static narration.

Run Pass 4: Chapter Closers. Flag endings that wrap up instead of pulling forward.

Run Pass 5: Dialogue Differentiation. Build a register chart for the main cast.

Run Pass 6: Language and AI-Tell Cleanup. Diagnose only. Do not rewrite unless I ask.

Run Pass 7: Continuity Audit. Track timeline, wounds, objects, names, settings, and character knowledge.

Run Pass 8: Front and Back Matter Check.

Run Pass 9: Final Assembly Checklist.

## Important

Do not ask the AI to edit the entire manuscript in one pass. Run the skill one pass at a time. The order matters.
