# Edit Like a Boss 8-Pass Crosswalk

Use this when Sylvia references the Edit Like a Boss tracker, an 8-pass audit, or pass labels like "P1 Quality" through "P8 Voice." The ELAB tracker is mostly covered by the existing Full-Stack Manuscript Polish workflow, but the names differ.

## Pass Mapping

| ELAB pass | What it means | Use our resource/pass |
|---|---|---|
| P1 Quality Check | Chapter-level readiness: natural prose, tone, outline adherence, arc, emotional progression, pacing, show/tell, ending, POV, heat/intensity, banned-list sweep | `chapter-by-chapter-edit` Chapter Quality Gate; Full-Stack Pass 6 for language cleanup |
| P2 Plot Holes | Logic gaps, missing causes, unsupported turns, unresolved setup/payoff, character decisions that do not make sense | Full-Stack Pass 1 Structure Check; `manuscript-editing` Developmental Assessment |
| P3 Timeline | Chronology, elapsed time, event order, travel time, wounds, object movement, character knowledge | Full-Stack Pass 7 Continuity Read-Through; `manuscript-editing` Continuity Audit |
| P4 Line Edit | Sentence-level prose quality, AI artifacts, show/tell, filter words, crutch phrases, weak mechanics | `manuscript-editing` Line Edit; Full-Stack Pass 6 Language / Voice / AI-Tell Cleanup |
| P5 Dialogue | Speaker differentiation, naturalness, tags/action beats, subtext, repeated yes/no/okay patterns | Full-Stack Pass 5 Dialogue Differentiation; `natural-dialogue` for deeper dialogue-only audits |
| P6 Heat / Action | Romance heat, intimacy escalation, action clarity, tension, physical continuity, scene choreography | `show-dont-tell-romance`, `show-dont-tell-fiction`, `manuscript-editing` intimate scene review, plus Full-Stack Pass 2 for missing beats |
| P7 Dev Edit | Big-picture pacing, scene function, arc satisfaction, genre promise, chapter purpose | Full-Stack Passes 1-2; `manuscript-editing` Developmental Assessment |
| P8 Voice Audit | Pen-name voice, character voice, consistency from first chapter to last, banned voice drift | Full-Stack Pass 6; `voice-doctor`; `ai-tell-eliminator`; `chapter-by-chapter-edit` voice integrity |

## ELAB Milestone Tests

Run these as stop/go checks, not as vague vibes:

- **After P3 / Continuity:** Can the whole book be read without a "that does not make sense" moment? If no, return to structure or continuity before line editing.
- **After P5 / Dialogue:** If dialogue tags are covered, can the reader still tell who is speaking? If no, run `natural-dialogue` or Full-Stack Pass 5 again.
- **After P7 / Developmental:** Would a target reader skip any scene? If yes, diagnose whether it needs a cut, merge, escalation, or replacement beat.
- **After P8 / Voice:** Does the final chapter sound like the first chapter and the same pen name? If no, run a voice audit before final assembly.

## Final Manuscript-Level Audit

After all chapter passes are complete, zoom out and record:

- Overall pacing: no sagging middle, overlong denouement, or rushed climax.
- Character arc resolution: logical and emotional resolution both land.
- Heat/tension progression: each scene builds on what came before; no sudden jumps or repetitive beats.
- Unresolved threads: setups, promises, clues, relationship beats, and subplots are paid off or intentionally held for series.
- Series setup: future-book hooks are planted naturally without hijacking this book's ending.

If any final audit item fails, do not call the manuscript ready. Send the book back to the relevant pass and log the reason.
